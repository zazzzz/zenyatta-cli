## Install

```bash
npm i -g zenyatta-cli
```

## Usage

```bash
zenyatta page.yml
```

with options

```bash
zenyatta page.yml -t zatlas -s zatlas -d output
```

## Options

```
-t, --template:    choose specific template, default: zatlas
-s, --style:       choose specific style, default: zatlas
-d, --dir:         choose output dir
```
